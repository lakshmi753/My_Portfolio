PROJECT DISCRIPTION

This project is a Todo List app, which is made using Html, Css and Javascript. In which we can schedule ours daily tasks. The app uses "local storage" to keep track of our tasks . The second top heading of the app keeps the count of total number of tasks. After that we have a 'text-area' where we can enter our desired tasks and along with it we have a button 'create task' which when clicked displays our task along with the date and time of the created task. We can edit and remove the task from the app using 'remove' button but only when the task is marked as "done". At the end we have a "clear list" button which gives us the option to clear the whole list at once but first makes sure wheather user really want to clear the list.

PROJECT DISCRIPTION

This project is a popular "Hangman-game", which everyone of us has definitely played once or more. This project is made using Html, Css and Javascript. Here I'm using self-create array of "words and hints" required for the game and randomly getting words along with there hints every time the game starts. Six chances were given for incorrect guess and with each incorrect guess our hangman image also get updated. After losing all the six chances there appears a overlay over which we can see a small pop-up having losing-gif, correct-word and a "play again" button, clicking which we can start the game again. The most facinating thing about this project is that in this project we are creating a keyboard using "String.fromCharCode()" method.

PROJECT DISCRIPTION

This project is a "Dice-Roll game", which is built using React. In React I used "useReducer hook" for all the calculations and state management. This game requires two players, at the starting Player-1 is the active player and roll the dice first by clicking on the "roll" button. The number on the dice become the "Current score" , player-1 can also "hold" with which "player-2" become active player and can roll the dice. If dice rolled "1" then also active palyer get switched. The player scoring 100 scores first wins the game. After the game get over only "new game" button appears clicking on which we can restart the game. I saw this game in Javascript but then tried it with React while I was learning "useReducer hook" as this app seems the right fit for practicing "useReducer" hook.

Have a look at this project by clicking on it.
